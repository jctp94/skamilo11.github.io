Estructuras de flujo y control

  • Estructuras if e if....else

    • Está estructura nos permite restringir una acción dependiendo si una condición se cumple o no.

    Decimos que una condición se cumple si su valor es true, la estructura es la siguiente: 

                 if(condicion){ 
                . 
                acción a realizar 
                . 
            }

    Existe una variación, o una función adicional a esta estructura llamada else que nos permite realizar una acción cuando la condición inicial no se cumple. Su estructura es así: 

                 if(condicion){
                . 
                acción a realizar 
                . 
            }else {
                . 
                acción a realizar 
                . 
            }

    Y si comprobamos una condición y si esta es falsa y ademas queremos comprobar otra condición podemos usar la estructura else if de esta manera 

                if(condicion){
                . 
                acción a realizar 
                . 
            }else if(otraCondicion){
                . 
                acción a realizar 
                . 
            }else {
                . 
                acción a realizar 
                . 
            }
  • Estructura switch case

    • Cuando queremos verificar varias condiciones, y estas solo dependen de una variable podríamos generar una serie de else if para cada caso pero esto es muy engorroso, existe otra estructura llamad switch case que nos permite solucionar este problema de una manera mas simple.

    Funcionan de la siguiente manera: 

                    switch(variable){
                    case valor1:
                    
                    .....código a ejecutar si la variable toma el valor1.....
                    
                    break;
                    
                    case valor2:
                    
                    .....código a ejecutar si la variable toma el valor2.....
                    
                    break;
                    
                    case valor3:
                    
                    .....código a ejecutar si la variable toma el valor3.....
                    
                    break;
                    
                    case default:
                    
                    .....código a ejecutar en caso de que no se cumpla.....
                    
                    break;
                }

    Podemos realizar una analogía entre el else y el default ya que ambos se activan si ninguno de los condiciones o casos se cumplen.

  • Estructura for

    • La estructura for nos permite realizar varias veces una acción hasta que un contador llegue a un valor determinado. Se escribe de la siguiente manera 

                for(inicializacion; condicion; actualizacion) {
                .
                acción a realizar
                .
            }

    La inicialización se refiere si bien a declarar la variable con la cual se va a tener control del ciclo, por ejemplo:
    var i = 0;
    o a indicar otra variable que haya sido declarada anteriormente

    la condición controla el ciclo preguntando cada vez que que se va a ejecutar la acción, si aún se cumple dicha condición. Ejemplo:
    i < 10;

    Finalmente la actualización cambia el valor de la variable con la que controlamos el ciclo. Ejemplo:
    i++;
    i--;
    i+=5;
    ...

    De esta manera un ejemplo de un for sería así: 
                for(var i = 0; i <= 10; i++) {
                .
                acción a realizar
                .
            }
  • Estructura for..in

    • Esta estructura derivada del for nos permite iterar de manera arbitraria de un objeto, En este caso vamos a mirar como funciona para recorrer un arreglo especificamente 

                var animales=["perro","gato","gallina","culebra","elefante"];
            
            for(var i = 0; in animales){
                console.log(animales[i]);
            }
    EL código anterior nos permitirá recorrer el arreglo e imprimir cada uno de sus elementos.

  • Estructura while

    • La estructura for nos permite realizar un accion repetitivamente pero siempre debemos decirle cuantas veces queremos que lo haga, entonces para ciertas situaciones esto es un poco rigido. Pero hay otra estructura que nos permite trabajar de una manera un poco mas flexible, el while que nos da la ventaja de no saber excatamente cuantas veces queremos que un código se repita, sino que simplemente se sigue repitiendo meintras se cumpla una condición que le demos. Su estructura es así: 

                     while(condicion){
                     .
                     ejecutar acción
                     .
                 }

    Donde condicion es la que se debe cumplir para que se siga ejecutando la acción que se encuentra dentro del while

    Una Variación de while es el do while que nos permite ejecutar por lo menos una vez la acción y luego de eso preguntar si se cumple la condición para que se vuelva a ejecutar. La estructura es la siguiente:

                     do{
                     .
                     ejecutar acción
                     .
                 }while(condicion);
