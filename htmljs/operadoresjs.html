Operadores

  • Asignación

    • Este es uno de los más usados y hemos trabajado con el bastante hasta el momento y se utiliza de la siguiente manera
    var asig,asig2;
    asig = 12;
    asig2 = asig;
    en ambos casos se está haciendo asígnación y el simbolo usado es "="

  • Aritmeticos

    • Dentro de este tipo de operadores podemos encontrar tres.
    Para incremento, se usa del siguiente modo:
    var a=5; ("Declaramos la variable")
    a++; "Aumentamos en uno su valor con lo que ahora a=6"

    Para decremento es un proceso similar:
    var a=5; ("Declaramos la variable")
    a--;
    "Decrementamos en uno su valor con lo que ahora a=4"

    Para calcular el residuo de una división se usa %:
    var a=13;
    var b=5;
    var c= a%b; "El valor de c será 3"

  • Lógicos

    • Negación
    Es representado con el signo de admiración(!) y devuelve el valor de verdad contrario al de la variable.
    Ejemplo:
    var cam= true;
    var neg= !cam; "En este caso el varo de neg sería false"
    Lo mismo pasaría al contrario
    var cam = false;
    var neg = !cam; "En este caso el valor de neg sería true"

    AND
    Es reprensentado por && y se usa con dos variables o terminos booleanos y devuelve true únicamente si ambos valores son true.
    Ejemplo:
    var a = true;
    var b = true;
    console.log(a && b); "Esto imprimirá true"
    var c = false;
    console.log(c && b) "Esto imprimirá false"


    OR
    Es representado por || y se usa igualmente con dos valores booleanos, devuelve true si alguno de los dos valores son true
    Ejemplo:
    var var a = true;
    var b = false;
    console.log(a || b); "Esto imprimirá true"
    var c = false;
    console.log(c || b) "Esto imprimirá false"

  • Matemáticos

    • Dentro de estos están los que se usan normalmente en todos los lenguajes de programación convencionales: "+","-","*","/".
    Ejemplo:
    var a= 10;
    var b= 4;
    console.log(a+b); "imprimirá 14"
    console.log(a-b); "imprimirá 6"
    console.log(a*b); "imprimirá 40"
    console.log(a/b); "imprimirá 2.5"
