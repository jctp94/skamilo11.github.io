Funciones

Cuando necesitamos repetir varias veces una instrucción dentro de una aplicación, el código se vuelve difícil de entender e inoficioso ya si queremos cambiar algo dentro de esta instrucción, nos tocaría modificar cada una de las veces que al repetimos. La solución ha dicho problema es una estructura denominada función. Una función es una serie de instrucciones que ejecutan una acción concreta y que puede ser facilmente reutilizada

Podemos definir una función de dos formas: declaración y espresión. Adicionalmente cuando la función desea retornar un valor es usada la palabara return para dicho proposito. 

                   //declaración.
                    function nombreFuncion(){
                        .
                        accion a realizar
                        .
                        return algunValor;
                    }
                    
                    //expresión
                    var nombreFuncion = function(){
                         .
                        accion a realizar
                        .
                        return algunValor.
                    }

Hay algunas veces que necesitamos que una función tome datos que le sean pasados por un ente externo. Para cumplir con estos objetivos se utilizan los "parametros", estos son valores que le llegan a una función para que esta pueda realizar alguna acción con estos. No todas las funciones requieren parametros ni necesariamente deben retornar un valor.

                    function sumar(x,y){
                       
                        return x+y;
                        
                    }

Es importante tener en cuenta que si en el ejemplo anterior no tuvieramos parametros no habría manera de sumar dos numeros cualquiera que le pasaramos a la función

Para llamar a la función simplemente se debe poner el nombre de esta y entre parentesis los parametros

                    var resultado = sumar(4,5);

Es importante recalcar que el númeor de parametros que se deben pasar a una función es el mismo que esta pide, si se le pasan mas o menos a diferencia de otros lenguajes javascript no muestra error por esto.
