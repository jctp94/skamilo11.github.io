Objetos

Un objeto en javascript así como en muchos lenguajes de programación puede ser comparado con un objeto de la vida real, este tiene ciertas carácteristicas unas funciones que puede realizar. En javascript a este tipo de características de un objeto se les conoce como propiedades

Existen varias formas de crear un objeto a continuación veremos algunas:
Mediante un constructor 

                       var vaso= new Object();

En este caso para asignarle propiedades al objeto se procede de la siguiente manera

                       vaso.material="vidrio";
                       vaso.tamano="mediano";

Para obtener el valor de una propiedad de un objeto se debe acceder a este de forma similar a como lo hacemos con arrays con arrays

                       var prop = vaso["material"];
                       var prop2 = vaso["tamano"];

Mediante notación literal:

                       var vaso = {
                           material: "vidrio",
                           tamano: "mediano"
                       }

Mediante una función constructora:

                       function Balon(deporte,color){
                           this.deporte = deporte;
                           this.color = color;
                       }

De esta manera si deseamos crear un objeto de tipo Balon lo hacemos así

                       var balonFutbol = new Balon(futbol,negro);

Una característica muy util de trabajar con objetos es que estos integran funciones de manera que un objeto no es solo un conjunto de propiedades sino también de funciones que nos permiten hacer determinadas acciones, las fucniones asociadas a un objeto se les llama metodos.
