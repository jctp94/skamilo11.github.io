Javascript

Sintaxis

En general la manera de escribir javascript es muy similar a la de otros lenguajes, pero tiene algunas particularidades:

● No es necesario especificar el tipo de dato que almacenará una variable.
●No es obligatorio poner ; al final de una linea de código aunque el hacerlo es considerado una "Buena práctica".
●No es obligatorio declarar las variables.
●No es necesario inicializar las variables

Variables

Las variables al igual que en otro lenguaje se emplean para lamacenar un dato, de manera que se pueda acceder a este en otro momento.

Para crear una variable en Javascript se usa la palabra reservada "var", y solo se usa cuando se va a declarar, es decir, al definir la variable.

var nombre = "pepito";

En este caso el nombre de la variable es nombre y su valor es "pepito"

Aunque en javascript está permitido no declarar variables (var algo = 2) antes de usarlas, es recomendable siempre hacerlo.

Tipos de variables

Aunque como ya se dijo anteriormente no es necesario definirle un tipo de dato a una variable, la forma en como les es asignado un valor depende del tipo de dato que se quiera almacenar

  1. Variables númericas

    2. En estas se pueden guardar valores tanto enteros como decimales, en el segundo caso se debe diferenciar con .(punto) la parte entera y la parte decimal.
    ejemplo:
    var numero=2;
    var decimal=2.1122;

  2. Variables con cadenas de texto

    3. Para guardar variables con texto es necesario poner este entre comillas simples('texto') o dobles("texto").
    ejemplo:
    var frase="Así se debe poner";
    var palabra='O también así';

  3. Boleanos
    4. La utilización de estas es muy sencilla, simplemente se declaran al igual que con las otras variables y se les asigna el valor booleano que deseemos.
    Ejemplo:
    var buenCurso= true; var nosRendimos=false;s
  4. Arrays
    5. Un array al igual que en otros lenguajes es una colección de variables, en el caso de javascript, pueden ser del mismo tipo o de difrente tipo.
    Declarar un array:
    var animales=["perro","gato","mico",true];
    var objetos=[2,"algo", false];
