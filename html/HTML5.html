Lo nuevo de HTML5

Entre los primeros cambios podemos evidenciar:

  • <!DOCTYPE html>: Declaración que define que el documento a realizar es de formato HTML5.

  • <meta charset="UTF-8">: Formato de codificación de caracteres a utilizar.

    • Tambien se han añadido una serie de elementos, entre los mas importantes:
  • Nuevas semánticas elementos como <header>, <pie de página>, <article> y <sección>.

  • Nueva forma de control de atributos como el número, fecha, hora, calendario y alcance.

  • Nuevos elementos gráficos: <svg> y <canvas>.

  • Nuevos elementos multimedia: <audio> y <video>.

    • asimismo se han eliminado algunos elementos utilizados en html4:
